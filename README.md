# MAFOR-Automatic-Decoding-Program

Welcome to my MArine FORecast automatic decoding program.

This program converts MAFOR codes to plain text which is easily parsed by the user.

To get started, download the program from the "Releases" Tab.

Once you open the program, it will prompt you to enter the MAFOR code. <br> <br>

<img width="1009" height="171" alt="image" src="https://github.com/user-attachments/assets/42a36d07-0fa6-4b02-a3b0-d9ef7a14adc8" /> <br>

Copy the FULL MAFOR code including the FQCN30... and ...END <br>

<img width="452" height="792" alt="image" src="https://github.com/user-attachments/assets/9a517d50-16a7-4932-8298-923f2172eacd" /> <br>

After pasting the entire code into the program and hitting "Enter", It will display the entire decoded message along with the region and time periods respectively. <br>
